# Lecture-Lens

Deployable static frontend prototype for Vercel.

## Files
- `index.html` — static entrypoint
- `lecturelens-prototype.js` — plain JavaScript demo logic (no build tools required)

## Run locally
Open `index.html` in a browser.

## Deploy on Vercel
1. Import this repository in Vercel.
2. Set framework preset to **Other** (or leave auto-detect).
3. No build command is required.
4. Deploy — Vercel will serve `index.html` as a static site.
